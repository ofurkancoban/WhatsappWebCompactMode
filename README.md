# WhatsApp Web Compact Mode

A revolutionary Chrome Extension that perfectly transforms WhatsApp Web into a sleek, infinitely resizable widget by safely restructuring its layout and overriding its width limitations.

## 🚀 Durum (Status)

**Geliştirme aşaması aktif olarak devam etmektedir. (Currently under active development)**

Bu eklenti; WhatsApp'ın arka planda çalışan sanal listeleme ve React satıriçi genişlik limitlerini akıllıca ezerek, tarayıcı pencerenizi istediğiniz kadar daraltmanıza olanak tanır. Özel bir üst bar (Top Bar) ve kusursuz çalışan **Avatar Bubble** (matematiksel olarak her koşulda ortalanmış profil resimleri) sistemi içerir. Daha fazla güncelleme için takipte kalın.
